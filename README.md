## Featured project

### SAP Navigator

A lightweight SAP navigation and troubleshooting tool built as a static web app.

The goal is simple: help SAP users quickly find the right transaction, table, flow or next step without digging through scattered documentation.

**Focus areas:**

- SAP transaction search
- Error and symptom decoding
- Common troubleshooting flows
- SAP table reference
- Favorites and local-first usage
- No backend, no database, no runtime AI cost

[View project](https://sapnav.dk) · [Repository](https://github.com/maurits2905/sap-navigator)
